Compare Commit / Membandingkan Commit :
1. Git Memiliki Fitur Untuk Membandingkan Antara Commit Dengan Commit Lainnya
2. Namun Jangan Sampai Salah Pengertian, Membandingkan Disini Adalah Membandingkan Snapshot Hasil Commit,
   Bukan Perubahan Yang Terjadi Antara Commit
3. Misal Pada Commit Sebelumnya Kita Pernah Menambah file3.txt, Namun Jika Kita Bandingkan Antara Commit Pertama
   Dan Terakhir (HEAD), Hasilnya Hanyalah Perbandingan Antara fil1 Dan file2, Tidak Ada file3
4. Hal Ini Dikarenakan Membandingkan Commit Bukanlah Membandingkan Perubahan Yang Pernah Terjadi,
   Melainkan Membandingkan Hasil Di Commit
5. Untuk Membandingkan Commit, Kita Bisa Gunakan Perintah :
   git diff hash1 hash2

Difftool :
1. Sebelumnya Kita Sudah Melakukan Pengaturan Menggunakan Visual Studio Code Untuk Melihat Diff
2. Jika Kita Ingin Menggunakan Visual Studio Code Untuk Melihat Perbedaan Antar Commit, Kita Bisa Gunakan Perintah :
   git difftool hash1 hash2

Kode Git Difftool Hash1 Hash2 :
git difftool e713386 HEAD
