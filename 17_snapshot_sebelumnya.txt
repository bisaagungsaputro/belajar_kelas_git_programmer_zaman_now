Snapshot Sebelumnya :
1. Git Memiliki Fitur Seperti Mesin Waktu, Dimana Kita Bisa Kembali Pada Snapshot Sebelumnya 
2. Kita Bisa Tentukan Kemana Tujuan Snapshot Kita Hanya Dengan Menggunakan Hash Commit
3. Cara Jika Kita Ingin Menuju Ke Snapshot Tertentu, Cukup Gunakan Perintah :
   git checkout hash
4. Jika Ingin Kembali Ke Paling Awal, Kita Bisa Gunakan Perintah :
   git checkout namabranch

Git Branch :
1. Materi Branching Akan Dibahas Pada Course Terpisah, Namun Secara Default Saat Kita Membuat Git Repository,
   Maka Secara Otomatis Git Akan Membuat Branch
2. Untuk Melihat Nama Branch Saat Ini, Kita Bisa Gunakan Perintah :
   git branch --show-current
