Versi Sebelumnya :
1. Kadang Kita Sering Mengalami Masalah Dengan File Yang Sudah Kita Commit Ke Repository
2. Git Memiliki Fitur Dimana Kita Bisa Melihat Versi File Pada Commit Sebelumnya
3. Saat Kita Ambil Versi Sebelumnya, File Pada Commit Tersebut Akan Berada Di Staging Index
4. Untuk Melakukannya, Kita Bisa Gunakan Perintah :
   git checkout hash --namefile
