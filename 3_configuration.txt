Configuration :
- Setelah Selesai Menginstall GIT, Hal Yang Pertama Kali Kita Lakukan Adalah Melakukan Konfigurasi.
- Yang Paling Utama Yang Perlu Kita Konfigurasi Diawali Adalah User Name Dan User Email.

=> Setiap Perubahan Yang Ada Pada Dalam Project GIT Akan Disisipkan Nama Dan Emailnya Supaya Tau Yang Mengubahnya Siapa.
Contohnya User Name : git config --global user.name "Bisa Agung Saputro"
Contohnya User Email : git config --global user.email "bisaagungsaputro@gmail.com"

Menggunakan Visual Studio Code :
- Agar Mempermudah, Kita Akan Menjadikan Visual Studio Code Sebagai Default editor Untuk Git Dan Default Diff Tool,
  Integrasi Git Ke Visual Studio Code => Ada Fitur Kita Ingin Melihat Perbedaan Dari Satu Versi Ke Versi Lainnya,
  Kita Bisa Menampilkan Perbedaannya Ke Visual Studio Code
  Perintahnya :
=> git config --global core.editor "code --wait"
=> git config --global diff.tool "default-difftool"
=> git config --global difftool.default-difftool.cmd "code --wait --diff \$LOCAL \$REMOTE"

- Melihat Seluruh Configuration :
=> git config --list --show-origin
