Blame :
1. Saat Membuat Kode Program Kita Ingin Tahu, Siapa Yang Menambahkan Baris Kode Program Tersebut Dan Apa Saja Yang Ditambahkan
2. Git Memiliki Fitur Yang Bernama Blame, Ini Digunakan Untuk Mencari Tahu, 
   Siapa Yang Menambah Perubahan Pada File Dan Juga Untuk Mengetahui Commitnya
3. Caranya Kita Bisa Gunakan Perintah :
   git blame namefile

Kode Git Blame :
git blame file1.txt
